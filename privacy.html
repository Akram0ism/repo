Privacy Policy for Glass New Tab Dashboard Last updated: December 2025 Overview Glass New Tab Dashboard is a browser extension that replaces the browser’s New Tab page with a customizable, glass-style productivity dashboard. We respect user privacy and are committed to transparency. The extension does not collect, sell, or share personal data. Data Collection Glass New Tab Dashboard does not collect any personal or sensitive user data. The extension does not: require user accounts use analytics or tracking services display ads transmit user data to external servers Local Data Storage The extension stores data locally in the user’s browser using chrome.storage. Stored data may include: bookmarks and bookmark groups widget layout and customization settings background image preferences user interface settings This data never leaves the user’s device. Permissions Explanation Tabs The tabs permission is used only for explicit user-initiated actions, including: retrieving the URL and title of the currently active tab when the user clicks the “Quick add bookmark” button detecting the currently playing media tab to enable media controls (play, pause, next, previous, volume) for supported websites The extension does not monitor browsing activity, track tab history, or access tabs without user interaction. Search The extension uses the Chrome Search API to perform searches only when the user submits a search query from the New Tab page. All searches respect the user’s default browser search engine. Search queries are not stored, logged, or modified by the extension. Content Scripts (YouTube & Spotify) Content scripts are used only to enable media controls on supported websites: YouTube YouTube Music Spotify Web Player The extension does not read personal content, messages, account data, or cookies from these websites. Host Permissions Host permissions are required only to enable specific features: api.open-meteo.com — to retrieve weather data openstreetmap.org — to display the map widget youtube.com, music.youtube.com, open.spotify.com — to enable media controls No remote JavaScript code is loaded or executed from these hosts. All extension logic is bundled locally. Geolocation Optional browser geolocation may be used only to show local weather information. Location data is used temporarily Location data is not stored Location data is not shared Third-Party Services The extension uses the following third-party services: Open-Meteo — weather data OpenStreetMap — map display No personal data is shared with these services. Remote Code Glass New Tab Dashboard does not use remote code. No external JavaScript is executed No eval or similar techniques are used All code is included in the extension package Data Sharing The extension does not sell, rent, or share user data with third parties. Changes to This Policy This Privacy Policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page. Contact If you have questions about this Privacy Policy, please contact the developer via the Chrome Web Store listing.
