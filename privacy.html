Privacy Policy for Glass New Tab Dashboard

Last updated: December 2025

Overview

Glass New Tab Dashboard is a browser extension that replaces the browser’s New Tab page with a customizable, glass-style productivity dashboard.

The extension respects user privacy and does not collect, sell, or share personal data.

Data Collection

Glass New Tab Dashboard does not collect any personal or sensitive user data.

  • No user accounts
  • No analytics or tracking
  • No ads
  • No data transmission to external servers

Local Data Storage

The extension stores data locally in the user’s browser using chrome.storage.

This may include:

  • Bookmarks and bookmark groups
  • Widget layout and customization settings
  • Background image preferences
  • User interface settings

All data remains on the user’s device and is never transmitted.

Permissions

Tabs

Used only for explicit user-initiated actions, such as adding the current tab to bookmarks or enabling media controls for supported websites.

Search

Search queries are executed using the Chrome Search API and respect the user’s default search engine. Queries are not stored or logged.

Content Scripts

Content scripts are used only to enable media controls on supported websites such as YouTube and Spotify. No personal data is accessed.

Host Permissions

Host permissions are required only for weather data (Open-Meteo), map display (OpenStreetMap), and media control functionality.

No remote JavaScript code is loaded or executed.

Geolocation

Optional geolocation is used only to display local weather information. Location data is not stored or shared.

Remote Code

This extension does not use remote code. All logic is bundled locally.

Data Sharing

No user data is sold, shared, or transferred to third parties.

Changes

This policy may be updated if functionality changes.

Contact

For questions, please contact the developer via the Chrome Web Store listing.
